# 5G-Protocol-Stack-Simulator

## Overview
This project simulates a simplified 5G/LTE User Equipment (UE) protocol stack.

Layers Implemented:
- NAS
- RRC
- PDCP
- RLC
- MAC
- PHY

## Features
- UE Authentication
- RRC Connection Setup
- Data Transmission
- Layer-wise Processing

## Tech Stack
- C++
- OOP
- Linux
- Makefile

## Build & Run

```bash
make
./simulator
